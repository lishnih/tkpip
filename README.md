# tkpip

### Installation

```sh
$ python -m easy_install <package.egg>
```
- or
```sh
$ py -3 -m easy_install <package.egg>
```

### After installation

- Now you can run the package from a commamd line:
```sh
$ run_tkpip.py
```
- or from a python environment:
```sh
from tkpip import tkpip
tkpip.main()
```
